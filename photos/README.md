# Photos for the Sunday Attendance dashboard

Drop image files in this folder. The dashboard's **Per Hub** page will pick them up automatically whenever you select that Governor (or Choir / Service) from the dropdowns. If a photo is missing, the dashboard shows a circle with the person's initials instead.

## File-naming rule (one line)

**Use the name exactly as it appears in the Google Sheet, plus an image extension.**

Spaces are fine — keep them. Match capitalisation. Allowed extensions: `.jpg`, `.jpeg`, `.png`, `.webp`.

| For | Example filename |
|---|---|
| Governor | `Lois Dorothy Nteful.jpg` |
| Choir | `LOVE IS PATIENT.jpg` *(uppercase, matches sheet)* |
| Service | `EXPERIENCE SERVICE.jpg` |

Photos are auto-cropped to a circle, so square / portrait pictures work best.

After you add files, commit + push and they appear live in ~1 min.
